# data

このディレクトリは、公開前の作業データを整理するための保管場所です。

## フォルダ構成

- `raw/`: 受け取り直後の元データ
- `work/`: 編集中データ
- `output/`: 公開候補、書き出し済みデータ、一覧表
- `archive/`: 古い版、差し替え前データ

## 現在のサブフォルダ

- `raw/source_images/`: 参照用の元画像
- `work/mappings/`: 画像名対応表や整理メモ
- `archive/unused_webp/`: 現在はHP未使用の書き出し画像

## 使い分け

`raw/`
- 元画像
- 元資料
- 受領ファイル一式

`work/`
- レタッチ途中
- リサイズ途中
- 並び順や掲載候補の作業メモ

`output/`
- 採用候補画像
- 掲載用の一覧
- 公開に使う前の最終確認データ

`archive/`
- 差し替え前の旧版
- 不採用だが保管しておきたい素材

## 運用ルール

- 公開中の `images/` は直接整理棚として使わない
- 元データは必ず `raw/` に残す
- 迷ったら `work/` に置き、確定後に `output/` へ移す
- 公開反映後に `change_log.md` を更新する
